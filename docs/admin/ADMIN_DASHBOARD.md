# EduOofa Admin Dashboard Design

## 1. Admin Dashboard Goals

The admin dashboard is an operations workspace for managing the college catalog, courses, applications, reviews, partner links, counselors, analytics, and revenue.

## 2. Navigation

- Overview.
- Colleges.
- Courses.
- Applications.
- Reviews.
- Counselors.
- Partner Links.
- Mentorship Bookings.
- Analytics.
- Revenue.
- Reports.
- Settings.

## 3. Overview Screen

Key metrics:

- Total colleges.
- Total courses.
- Active students.
- New leads.
- Open applications.
- Approved reviews.
- Pending reviews.
- Mentorship bookings.
- Gross revenue.
- Conversion rate.

Operational queues:

- Applications needing action.
- Reviews awaiting moderation.
- Colleges with stale data.
- Failed payment events.
- Counselor workload alerts.

## 4. College Management

Capabilities:

- Create, edit, archive, and publish colleges.
- Manage campuses, addresses, ownership, approvals, accreditation, rankings, media, facilities, hostels, scholarships, admission details, and SEO metadata.
- Manage course offerings by mode: online, offline, distance.
- Manage partner links and sponsored visibility.
- Validate content completeness before publishing.

## 5. Course Management

Capabilities:

- Create course master records.
- Link courses to colleges.
- Manage specializations, fees, duration, eligibility, entrance exams, seats, application deadlines, salary statistics, and placements.
- Track data freshness.

## 6. Application Management

Capabilities:

- View all applications.
- Filter by status, college, course, counselor, source, date, and payment status.
- Assign counselor.
- Update status.
- Add admin notes.
- Export CSV.

Core statuses:

- Draft.
- Submitted.
- Under Review.
- Documents Required.
- Sent to Partner.
- Accepted.
- Rejected.
- Withdrawn.

## 7. Review Moderation

Capabilities:

- View pending, approved, rejected, and flagged reviews.
- Moderate text and rating fields.
- Mark verified student reviews.
- Feature high-quality reviews.
- Record moderation reason.

## 8. Analytics

Dashboards:

- Traffic analytics.
- Search analytics.
- College page views.
- Course page views.
- Shortlist analytics.
- Comparison analytics.
- Application funnel.
- Booking funnel.
- AI recommendation performance.
- Partner college performance.

## 9. Revenue Tracking

Revenue sources:

- Mentorship bookings.
- Application fees.
- Partner leads.
- Sponsored college links.
- Featured placements.
- Report downloads, if monetized later.

Metrics:

- Gross revenue.
- Net revenue.
- Refunds.
- Failed payments.
- Revenue by college.
- Revenue by counselor.
- Revenue by source.
- Monthly recurring partner revenue, if applicable.

## 10. Admin UX Requirements

- Dense, table-first operational screens.
- Bulk actions where safe.
- Server-side filtering and pagination.
- Clear publish status.
- Audit visible on sensitive records.
- Export support for key tables.
