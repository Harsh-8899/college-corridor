# EduOofa Roadmap

## Phase 1: Planning and Architecture

Deliverables:

- PRD.
- System architecture.
- User roles and permissions.
- Prisma database schema design.
- API architecture.
- Admin dashboard design.
- Counselor CRM design.
- Mentorship booking design.
- AI recommendation engine design.
- Project folder structure.
- Implementation plan.

Exit criteria:

- Stakeholder approval on scope.
- Data model approved.
- API boundaries approved.
- Phase 2 priorities confirmed.

## Phase 2: Foundation Build

Scope:

- Initialize Next.js 15 project.
- Configure TypeScript, Tailwind CSS, Shadcn UI.
- Configure Prisma and PostgreSQL.
- Configure NextAuth.
- Implement RBAC helpers.
- Implement base layouts.
- Implement shared UI components.
- Add seed data for development.

Exit criteria:

- App boots locally.
- Auth works.
- Prisma migrations run.
- Basic role routing works.

## Phase 3: College Catalog and Student Workflows

Scope:

- College listing and filters.
- College profile pages.
- Course pages.
- Saved colleges.
- College comparison.
- Student profile and preferences.
- PDF report generation.

## Phase 4: Applications, Reviews, and Admin

Scope:

- Application workflow.
- Admin college/course management.
- Review submission and moderation.
- Partner links.
- Basic analytics.

## Phase 5: Counselor CRM and Mentorship

Scope:

- Counselor dashboard.
- Student assignments.
- CRM notes, tasks, call logs.
- Availability.
- Booking system.
- Razorpay payments.
- Notifications.

## Phase 6: AI Recommendations

Scope:

- Recommendation scoring service.
- AI explanation generation.
- Recommendation history.
- Student recommendation dashboard.
- Counselor recommendation tools.

## Phase 7: Production Hardening

Scope:

- Security audit.
- Performance optimization.
- Observability.
- Rate limits.
- Backups.
- Admin audit logs.
- Load testing.
- SEO optimization.
- Production deployment.
